# AI Auto Article Gemini Pro (WordPress Plugin)

A powerful WordPress plugin designed to automate high-quality article generation using the Gemini API. This plugin focuses on content efficiency and advanced Search Engine Optimization (SEO).

## 🚀 Key Features

- **Gemini API Integration:** Generates natural, informative, and context-aware articles automatically.
- **Advanced SEO Optimization:** Built with a focus on entity optimization and Semantic Keyword (LSI) usage.
- **Auto-Tagging System:** Automatically generates relevant tags to improve internal linking and site structure.
- **Clean Architecture:** Optimized for technical site health and proper heading hierarchy (H1, H2, H3).
- **Modern UI:** Features a user-friendly admin interface for easy configuration.

## 🛠️ Tech Stack

- **PHP:** Core WordPress plugin logic and hooks.
- **JavaScript:** Responsive dashboard interactions.
- **SQL (WordPress Database):** Efficient metadata management and configuration storage.
- **Gemini Pro API:** The engine behind the content generation.

## 📋 Installation

1. Download or clone this repository.
2. Upload the plugin folder to the `/wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin through the **Plugins** menu in the WordPress Dashboard.
4. Navigate to the plugin settings and enter your **Gemini API Key**.
